# CLAUDE.md

## What this project is

This is a data-driven response to the conservative argument that women have been "taking over" the labor market since the 1990s, pushed by government policy. The specific piece it responds to is Inez Stepman's "The Myth of the Independent Girlboss" in First Things, which argues women's employment has been artificially boosted by policies dating to the early 1990s (the Civil Rights Act of 1991, federal student loans, immigration) and that women are colonizing industries through tipping-point dynamics.

The analysis shows three things:

1. **Women's labor force participation plateaued in the mid-1990s.** Prime-age (25-54) women's LFPR rose steadily from the 1940s through ~1999 and has been flat since. The policies named in the First Things piece coincide with the plateau, not an acceleration.
2. **Industry gender composition is almost entirely persistent.** The correlation between an industry's women's share in 1990 and its share in 2025 is 0.955 across 209 sub-industries. A panel AR(1) model estimates rho = 0.92, strongly rejecting explosive dynamics. Only 2 of 209 industries show statistically significant explosive behavior.
3. **The small aggregate increase is a composition effect.** A shift-share decomposition shows the within-industry component is slightly negative (industries became marginally more male). The entire aggregate increase in women's employment share comes from the economy shifting toward sectors that were already female-dominated (health care, education), not from women entering new industries.

The main output is `03_figures.qmd`, which currently renders all figures without prose (the blog text has been removed).

## Pipeline

Run these in order. Steps 1 and 2 are data pulls; step 3 is the analysis.

| # | File | What it does | Requires |
|---|------|-------------|----------|
| 1 | `01_pull_ces_data.R` | Pulls Current Employment Survey data via the `tidyusmacro` R package. Saves `data/ces_women.rds`. | R, `tidyusmacro` package |
| 2 | `02_pull_cps_data.py` | Downloads CPS Basic Monthly microdata from IPUMS API for 1992, 2019, 2024. Saves a `.dta` file to `../data/` (one directory up). | Python, `ipumspy`, `IPUMS_API_KEY` |
| 3 | `03_figures.qmd` | **Main output.** Loads CES, CPS, firm-size, and BLS API data. Produces Figures 1-6, Table 1, and Figure A1. Exports retina PNGs to `graphic/`. | R, Quarto, `BLS_KEY` in `.Renviron` |

Supplementary analysis files (not part of the main pipeline, but use the same data):

| File | What it does |
|------|-------------|
| `04_detailed_ces_analysis.qmd` | Deeper CES industry analysis: histograms of women's share distribution over time, scatter plots, AR(1) panel and industry-by-industry estimates, Duncan dissimilarity index, transition matrices, employment-weighted measures, near-parity industry trajectories. ## Data

### Included in the repo

- **`data/ces_women.rds`**: CES annual data (M13 averages). Columns: `year`, `industry_name`, `supersector_name`, `industry_display_level`, `total`, `women`, `pct_women`, `diffusion_index`. Generated by `01_pull_ces_data.R`.
- **`data/firm_size.rds`**: QCEW Business Dynamics firm-size distribution. Used in Figure 6. No generation script in repo; treat as a manually prepared input derived from `data/table_f.txt`.
- **`data/table_f.txt`**: Raw BLS QCEW table of private-sector employment by firm size class, 1993-2025. Source for `firm_size.rds`.

### External (must be fetched)

- **CPS microdata** (`../data/cps_00040.dta`): ~250MB IPUMS CPS extract. Lives one directory up, not inside the project folder. Generated by `02_pull_cps_data.py`. Requires an IPUMS account and API key.
- **BLS API**: `03_figures.qmd` pulls prime-age LFPR series (LNS11300061, LNS11300062) live at render time. Requires `BLS_KEY` in `~/.Renviron`. The API limits requests to 20-year windows, so the code batches into four calls (1948-1967, 1968-1987, 1988-2007, 2008-2025). If the API call fails, Figure 1 is skipped gracefully via `tryCatch`.

### API keys

Scripts expect these environment variables (typically set in `~/.Renviron`):

- `BLS_KEY` -- for BLS API calls in `03_figures.qmd`
- `IPUMS_API_KEY` -- for CPS data download in `02_pull_cps_data.py`

Keys are never hardcoded. Do not print or expose them.

## Analytical sample definitions

### CES industry sample

- 209 "diffusion index" industries (`diffusion_index == TRUE`), private sector only.
- These are the ~250 sub-industries BLS uses to compute its payroll diffusion index. Of 250, 209 report women's employment. The missing 41 are mostly small manufacturing industries, which would reinforce the findings (they are male-dominated).
- M13 (annual average) values, 1990-2025.
- `pct_women = women / total` is the key variable.

### CPS sample

- IPUMS CPS Basic Monthly, years 1992, 2019, 2024 (all 12 months each).
- Armed forces dropped (`CLASSWKR != 26`).
- Ages 16+ with `LABFORCE > 0` for the main analysis; ages 20-29 for Table 1.
- Education split: `EDUC >= 111` = college (BA+).
- LFPR = `LABFORCE == 2` / (`LABFORCE %in% c(1,2)`), weighted by `WTFINL`, averaged across months within each year.

### BLS API series

- `LNS11300061`: Men 25-54 LFPR, seasonally adjusted monthly.
- `LNS11300062`: Women 25-54 LFPR, seasonally adjusted monthly.
- Annual values are computed as the mean of 12 monthly observations.

## What each figure shows

**Figure 1** (LFPR time series): Prime-age men's and women's LFPR from 1948 to present. The key claim: women's LFPR plateaued around 2000 and has been flat for 25 years. Data: BLS API.

**Figure 2** (CES women's share): Women as a share of total nonfarm payroll employment, 1964-2025. Rose from 32% to 48% by 1995, then barely moved to 50% by 2025. Data: CES total nonfarm.

**Table 1** (LFPR by age/education): LFPR for ages 20-29 by sex and college status in 1992, 2019, 2024. Tests whether the aggregate plateau masks divergence among young cohorts. It does not. Data: CPS microdata.

**Figure 3** (three-sector panels): Employment levels and women's share for manufacturing, professional/business services, and health care/education. Shows that women's share within each major sector is flat or declining, even as total employment shifts dramatically. Data: CES supersectors.

**Figure 4** (scatter + quintiles): Panel A scatters each industry's 1990 women's share against its 2025 share (r = 0.955). Panel B shows mean change in women's share by 1995 starting quintile. The most female-dominated industries lost female share; male-dominated ones gained slightly. Mean reversion, not tipping points. Data: CES diffusion index industries.

**Figure 5** (shift-share decomposition): Decomposes the aggregate change in women's employment share (1999-2025) into within-industry, between-industry, and interaction components. The within component is negative. The entire increase comes from between-industry reallocation. Data: CES diffusion index industries.

**Figure 6** (firm size): Share of private-sector employment by firm size (under 50, 50-999, 1000+), comparing 1993 to 2025. Shows the shift toward larger firms, offered as an alternative explanation for changes in workplace culture. Data: `data/firm_size.rds` (QCEW Business Dynamics).

**Figure A1** (AR(1) distribution): Histogram of industry-by-industry AR(1) coefficients on women's share. Tests the tipping-point hypothesis formally. The median coefficient is ~0.89, well below 1. Almost no industries show explosive dynamics. Data: CES diffusion index industries.

## Where to challenge specific claims

If you are reviewing this analysis critically, here are the places to push:

- **"Plateaued in the mid-1990s"**: Figure 1 and Figure 2. Check whether the plateau date is sensitive to the age window (25-54 vs. 25-64 vs. 16+) or to using employment rates instead of LFPR. `05_cps_exploration.qmd` has some of this.
- **"r = 0.955"**: Figure 4A. This is a levels-on-levels correlation, which will be high mechanically if shares are persistent. The more informative test is the AR(1) in Figure A1 and the quintile analysis in Figure 4B. You could also check whether weighting by employment changes the correlation, or whether results differ using 5-year or 10-year changes instead of 35-year endpoints.
- **"Shift-share shows it's all between"**: Figure 5. The base year (1999) matters. Try 1990 or 1995. The interaction term is small here but could matter with different endpoints. The decomposition is exact (no residual), so you can verify it sums correctly.
- **"209 industries"**: The missing 41 industries (mostly manufacturing) could matter if they behave differently. `04_detailed_ces_analysis.qmd` has a coverage check. You could also argue the diffusion index industries are too coarse and that within-industry shifts are hidden inside broad categories.
- **"AR(1) rejects explosive dynamics"**: Figure A1 and the `ar1-test` chunk. The panel FE estimate pools all industries. You could argue that a few specific industries matter more than the average, or that 35 years of annual data is short for detecting slow tipping dynamics. The Nickell bias in short panels with fixed effects pushes rho down, though with T=35 this is modest.
- **"Firm size explains workplace culture change"**: Figure 6. This is offered as a hypothesis, not a tested causal claim. The data shows the structural shift but does not establish a link to workplace culture.
